什么是JavaScript

历史回顾

1995年，网景公司的布兰登·艾奇（Brendan Eich）开发了一款名为LiveScript的脚本语言。后因当时Java的火热，网景公司又将其更名为JavaScript。
1996年，微软的IE3发布，其中包含了名为的JScript的JavaScript实现。
1997年，JavaScript 1.1作为提案被交给欧洲计算机制造商协会（ECMA），由第39技术委员会（TC39）进行标准化工作。ECMA-262，也就是ECMAScript这个新的脚本语言标准。
1998，国际标准化组织（ISO）和国际电工委员会（IEC）也将ECMAScript采纳为标准，自此以后，各家浏览器均以ECMAScript作为自己JavaScript实现的依据。
ECMA-262最新版本为第12版，发布于2021年6月。

JavaScript的实现

JavaScript是一种在客户端运行的脚本语言，不需要编译，由js解释器逐行解释并执行。
某种程度上JavaScript和ECMAScript基本上可以视作同义词，但JavaScript远不止ECMA-262定义的内容。完整的JavaScript实现包含了三个部分：核心（ECMAScript）、文档对象模型（DOM）和浏览器对象模型（BOM）。

  • ECMAScript：是ECMA-262定义的语言。web浏览器只是ECMAScript实现可能存在的一种宿主环境。
  • DOM：文档对象模型（Document Object Model），提供了于网页内容交互的方法和接口。
  • BOM：浏览器对象模型（Browser Object Model），提供了用于访问和操作浏览器窗口的API。

语言基础

语法

  • JavaScript中一切都区分大小写。
  • 标识符第一个字符必须是字母、下划线_或者美元符号$，第二个字符开始可以有数字。
  • 标识符最好采用小驼峰式写法，即第一个单词首字母小写，后面每个单词首字母大写。
  • 严格模式"use strict"，添加在整个脚本文件的顶部，会对整个脚本文件都启用严格模式。
  • 语句以分号;结尾，代码块则由{}标识，//和/*...*/分别表示单行和多行注释。

变量

JavaScript变量是松散类型的，可以用于保存任何类型的数据。
声明变量有三个关键字var、let和const，其中let和const在ES6以后才可以使用。

var

  • 使用var在函数内部定义的变量，会成为函数的局部变量。
  • 使用var在函数内部定义变量时，所有变量声明会提升（hoist）到函数顶部。
  • 反复多次使用var声明同一个变量也不会有问题。

const

const的行为与let基本相同。
唯一的重要区别：const必须在声明的同时初始化变量且之后不允许修改。
如果const变量引用的是一个对象，修改对象内部的属性不违反const限制。

建议：不要使用var。优先考虑const，对需要修改的变量再使用let。

数据类型

JavaScript中有8种基本的数据类型，其中包含了7种原始类型和1种引用类型。
但JavaScript是一种动态类型（Dynamically Typed）的语言，意味着虽然有数据类型，但定义的变量并不会在定义后被限制为某一种数据类型。

typeof操作符会返回参数的类型。

String

表示零个或多个16位unicode字符序列。JavaScript中字符串必须写在引号内。
引号可以用双引号、单引号或者反引号，双引号和单引号都是简单的引用，使用上几乎没有什么区别。
反引号是功能扩展的引号，允许通过将变量或者表达式包装在${...}中来将它们嵌入到字符串。
JavaScript中字符串是不可变的。要想修改变量中的字符串值，必须先销毁原始字符串，再将含新值的字符串保存到变量中。

Number

Number类型使用IEEE 754格式表示整数和浮点数（某些语言中的双精度值）。
整数除了十进制以外，也可以使用八进制或十六进制，但是在运算时都会被当作十进制进行运算。
定义浮点数必须包含小数点.，且后面必须有一个不为零的数字，否则JavaScript会将其转换为整数。
涉及到JavaScript中浮点数，更为经典的一个问题是0.1 + 0.2为什么不等于0.3。

除了常规的数字，还有特殊的数值也属于Number类型。

  • Infinity 代表无穷大，可以用数字除以0获得它。
  • NaN 代表一个计算错误，是Not a Number的缩写，对NaN的进一步数字运算都只会返回NaN。

BigInt

BigInt类型是最近被添加到JavaScript中的，用于表示任意长度的整数。
通过在整数字段的末尾添加n来创建一个BigInt值。

Boolean

Boolean类型只包含两个值true和false。
true代表真值，false代表假值。

Undefined

Undefined类型的只有一个值，即特殊值undefined，其含义是“未被赋值”。
一个变量如果已经声明，但是没有被赋值，则它会被默认赋予undefine值。
永远不要显式赋值undefined，并且建议在声明变量的同时初始化变量。

Null

Null类型同样也只有一个值，即特殊值null。
逻辑上讲，null值表示一个空对象指针，因此typeof()会返回object。
与不要显式赋值undefined不同，在定义将来要保存对象值的变量时间，建议用null来初始化。

Symbol与Object

Symbol类型也是最近被添加到JavaScript中的，它是一个原始类型。
Symbol类型用于创建对象的唯一标识符，确保对象不会发生属性冲突的危险。
Object类型代表了一组数据和功能的集合。

运算符

这部分的内容主要参考W3School的ECMAScript运算符
运算元：运算符操作的对象，也称为参数。
如果一个运算符对应一个操作对象，则称为一元运算符。如果它对应两个操作对象，则称为二元运算符。

  • 一元运算符：自增++、自减--、一元加减+/-
  • 位运算符：按位非~、按位与&、按位或|、按位异或^
  • 逻辑运算符：逻辑非!、逻辑与&&、逻辑或||
  • 乘性运算符：乘法*、除法/、取余%
  • 加性运算符：加法+、减法-
  • 关系运算符：小于<、大于>
  • 等性运算符：等号==、非等号!=、全等号===、非全等号!==
  • 条件运算符：variable = boolean_expression ? true_value : false_value;
  • 赋值运算符：=
  • 逗号运算符：,

优先级：运算符优先级参见优先级表，一般一元运算符优先级高于二元运算符。

函数

函数基础

函数是程序的主要”构建模块”，用于该段代码需要多次调用的时候。

声明与命名

函数的封装：将一个或多个功能通过函数的方式封装起来，对外只提供一个简单的函数接口。
函数命名：函数是行为，因此命名尽可能用动词前缀来进行。

变量与参数

  • 局部变量：在函数中声明，只在该函数内部可见。
  • 外部变量：函数对外部变量拥有全部的访问权限，也可以对外部变量进行修改。
  • 如果函数内部声明了同名变量，则函数会遮蔽外部变量，优先使用内部变量。

参数与默认值

  • 函数调用时，实参(argument)’Ann’与’Hello!’被复制到了局部变量形参(parameter)from与text。
  • 声明函数时，列出它们的参数(parameters);调用函数时用它们传递参数(arguments)。
  • 如果函数被调用，但是参数(argument)未被提供，那么相应的值会变成undefined。

返回值

  • 函数可以将一个值返回到调用代码中作为结果。
  • 指令return可以在函数中任意位置，当执行到达时，函数停止。一个函数中可以有多个return。
  • return可以没有返回值，但是会导致函数立即退出。

函数表达式

在JavaScript中，函数是一种特殊的值。
函数的创建有两种方式，一种是函数声明，另一种为函数表达式。

因为函数是一种特殊的值，所以可以像使用其他的值一样使用它。

回调函数：将函数作为值来传递

函数声明与函数表达式的区别

  • js引擎运行js代码时先预解析，然后再进行代码执行。
    1. 预解析时，变量和函数声明会被提升到当前作用域的最前面，但赋值和调用并不进行。
    2. 代码执行时，按照书写顺序从上往下执行。
  • 因此，函数声明在被定义之前就可以被调用，而函数表达式只能在代码执行到达时被创建，并且仅从那一刻起可用。
