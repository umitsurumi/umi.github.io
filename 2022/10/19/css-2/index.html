CSS补充

CSS单位

CSS中的单位可以分为两种：绝对单位和相对单位。

  1. 绝对单位
绝对单位 说明 等价代换
cm 厘米 1cm = 96px/2.54
mm 毫米 1mm = 1/10th of 1cm
in 英寸 1in = 2.54cm = 96px
pt 点 1pc = 1/6th of 1in
px 像素 1px = 1/96th of 1in
  • 绝对单位一般用于传统平面印刷，实际网页开发中使用较少。
  • px像素值在MDN中被归类为绝对单位。
  1. 相对单位
相对单位 说明
em 在font-size中是相对于父元素字体大小，在其他属性中使用是相对自身字体大小
rem 相对根元素字体大小
% 相对父元素的百分比
vw 相对可视区域宽度的百分比
vh 相对可视区域高度的百分比

CSS精灵图

精灵图(CSS Sprites)，又称CSS雪碧图。主要使用场景为页面的背景。
目的：减少服务器接收和发送请求的次数，提高页面加载速度。
实现：将多张小图片拼合成一张大图片，使用时通过background-position来展示需要使用的那部分图片。

缺点：使用较麻烦，每一种图片都需要分别设置并调整位置。

CSS字体图标

字体图标(iconfont)是一种特殊的字体，它是一个“编码-图形字体”的映射表。因此，虽然它展示出来是图标，但是可以像字体一样去使用。
优点：轻量级，灵活性，兼容性。适用结构样式较简单的小图标。

表单样式

  • 轮廓线(outline)：

    • outline与border很类似，也有outline-style、outline-width与outline-color属性。
    • outline不占据空间，绘制与元素四周。默认情况下outline用于:focus样式。
    • 使用outline:0或者outline:none去除默认的outline。

  • resize属性

    • resize属性指定一个元素，允许用户调整大小。
    • 禁止拖拽调整元素尺寸(如textarea)使用resize:none。

  • 列表项符号(list-style-type)

    • ol可取值：decimal | lower-roman | upper-roman | lower-alpha | upper-alpha
    • ul可取值：disc(实心圆) | circle(空心圆) | square(正方形) | url(自定义图片)
    • 去除默认值：list-style-type:none

  • user-select属性
    user-select属性用来控制用户能否选中文本。

    属性值 说明
    none 元素及其子元素的文本不可选中
    all 包含该元素的最顶层元素也会被选中
    text 用户可以选择文本
    contain 允许在元素内选择，但是选区限制在该元素的边界之内
    auto 取决于父元素的user-select和元素本身情况

CSS3新特性

CSS3是最新的CSS标准，从CSS3开始，CSS被划分为众多模块，这些模块彼此独立并按照各自进度来进行标准化。

选择器

属性选择器

属性选择器在CSS2中引入，在CSS3中得到了进一步的扩展，它的使用语法如下：

选择器 说明
[attr] 选中具有attr属性的元素
[attr=val] 选中具有attr属性，且值为val的所有元素
[attr~=val] 选中具有attr属性，且值包含val的所有元素
`[attr =val]`
[attr^=val] 选中attr属性值以val开头的所有元素
[attr$=val] 选中attr属性值以val结尾的所有元素
[attr*=val] 选中attr属性值包含子串val的所有元素

伪类与伪元素

除了:nth-of-child(n)与nth-of-type(n)系列，CSS3还提供了几个较为常用的新选择器：

  1. :root
    :root选择器匹配文档根元素，在HTML中则始终是html元素，配合CSS变量可以在文档中重复使用。

  2. ::after与::before
    在CSS3规范中，用:和::来对伪类和伪元素进行区分。
    ::after表示在选中元素之后插入内容，::before表示在选中元素之前插入内容，两者必须搭配content来进行使用。

    优点：减少DOM节点数，优化页面结构。
    缺点：不利于SEO，无法使用js控制。

盒子属性

box-sizing

CSS3中可以通过box-sizing属性来指定盒子模型。

  • box-sizing: content-box，默认值，标准盒子模型。
  • box-sizing: border-box，IE盒子模型，又称怪异盒子。
    它们区别如下图所示：

在标准盒子中，盒子宽度 = width + padding + border;
在怪异盒子中，盒子宽度 = width，而内容区空间会被padding和border挤进去。

border属性

CSS3的边框新增了border-image与border-radius属性。

  • border-image属性用来设置图像作为围绕元素的边框。

    属性 取值 说明
    border-image-source 路径 选择图片路径作为绘制边框来源
    border-image-slice 像素值 规定图像边框图像的内偏移
    border-image-width 图像宽度 多个值时应用顺序与内外边距相同
    border-image-outset 像素值 定义边框图像可超出边框盒的大小
    border-image-repeat `stretch repeat

    border-image可以进行简写：

    • 顺序为url() slice width outset repeat
    • 默认值为none 100% 1 0 stretch

  • border-radius

    • border-radius: 30px圆角；border-radius: 30px/20px椭圆角。
    • border-radius可以给所有元素设置圆角，当设置border-radius: 50%时，就得到了一个圆。
    • border-radius实际上是指定了一个圆(或椭圆)与外边框进行交集，从而形成的圆角效果。
    • border-radius也是对四个边角的简写，同样也可以四个角分别指定。

过渡与动画

过渡(transition)

transition属性允许在不使用脚本或添加动画的情况下，平滑地改变属性值。
要应用过渡属性，需要指定添加效果的属性，并且指明效果持续时间。

属性值 说明
transition-property 规定过渡效果应用的CSS属性
transition-duration 规定过渡效果持续时间(秒或毫秒)
transition-delay 规定过渡效果的延迟(秒)
transition-timing-function 规定过渡效果速率曲线

类似的，transition也可以对这四个属性进行简写:

  • 只有具有中间值的属性才可以应用过渡效果，如display属性不能实现过渡。

  • transition-属性是即时生效的，如果写在:hover里面，则移入有效果，移出无效果。

  • transition-timing-function各关键词参考见MDN。

    关键词 说明
    linear 线性应用过渡效果
    ease 由快到慢，逐渐变慢
    ease-in 速度越来越快
    ease-out 速度越来越慢
    ease-in-out 先加速后减速
    cubic-bezier(n,n,n,n) 在cubic-bezier函数中定义效果

动画(animation)

动画可以设置多个节点来控制一个或一组动画效果。相较过渡，动画可以实现更多变化、更多控制、自动播放等。

  • 动画不需要事件触发，过渡需要。
  • 过渡只有一组(开始与结束)关键帧，动画可以设置多个。

同样，animation也可以对动画属性进行简写:
animation: name duration timing-function delay iteration-count direction fill-mode play-state
