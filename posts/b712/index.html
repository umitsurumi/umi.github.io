# 栈

栈是一种遵循后进先出（Last in First out,LIFO）的数据结构。
进行出栈和入栈操作的一段称为栈顶，不能进行操作的一段称为栈底。
当有nnn 个不同的元素入栈，出栈的元素共有1n+1C(n2n){1\over n+1} C\binom{n}{2n}n+11​C(2nn​) 种不同的排列。

# 栈的应用

栈的实际应用非常广泛，在实际应用中，它可以存储访问过的任务或路径、撤销的操作，像是进制转换、表达式求值都可以用到栈，函数的递归调用利用到了栈。

# 进制转换

十进制转二进制采用 “除二取余，逆序排列” 的方法。
具体的说，就是将一个十进制数除以二，然后得到商和余数，再把商用来除以二，直到商为 0。然后将这些余数逆序作为二进制数的高位到低位。
这里这个逆序，即符合栈后进先出的特性。

# 队列

队列是一种遵循先进先出（First in First out,FIFO）的数据结构。
队头是只允许进行删除操作的一端，与之对应的操作称为出队；队尾是只允许进行插入操作的一端，与之对应的操作称之为入队。

# 链表实现

链式队列的实现同样也只需要在之前单链表的基础上稍作修改。
为了符合栈 FIFO 的特性，单链表可以使用尾插法创建，这样就保证先创建的结点在链表的头部，而后创建的结点在链表尾部。
需要注意的是，除了头指针，还需要一个尾指针来实现入队的操作。

# 队列的应用

队列的实际应用也很广泛，诸如操作系统的 I/O 调度、多用户的资源竞争等场景。
浏览器中的事件循环机制，也使用到了队列。

# JS 中的数组

不同于其他语言中的数组，JavaScript 中的数组实际上也是一种特殊的对象，可以使用 Array instanceof Object 证明。 array[index] 实际上也是对象 object[key] 的语法。
但是与一般的对象不同，JavaScript 引擎对数组进行了一些特殊的优化，而其中的元素也尽可能一个接一个存储在连续的内存区域内，因此当我们将数组视作一个有序集合的特殊结构来使用时，它的效率要高于普通对象。
