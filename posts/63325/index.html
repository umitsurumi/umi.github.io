# Flex 布局

Flex 布局是一种用于按行或按列布局的一维布局方法，元素可以膨胀填充额外空间，也可收缩适应更小空间。
使用 display:flex 或 display:inline-flex 来指定一个 flex 容器。

  • 主轴 (main axis) 是沿着 flex 元素放置方向延伸的轴线。
  • 交叉轴 (cross axis) 是垂直于 flex 元素放置方向的轴线。
  • flex 容器 (flex container) 是指设置了 display:flex 的父元素。
  • flex 项 (flex item) 是在 flex 容器中的元素。

# align-content

  • align-content 定义了多根基线沿交叉轴的对齐方式，只有一根基线时该属性不起作用。
  • 当 flex 容器设置 flex-wrap:wrap ，且 flex 项进行了换行后， align-content 属性才会生效。
  • align-content 取值为 flex-start|flex-end|center|space-between|space-around|stretch 。

# flex 项

flex 项可设置以下几种属性：

# flex

  • flex 是对 flex-grow flex-shrink flex-basis 三个属性的缩写。
    • flex-grow ，定义项目放大比例，默认值为 0，有剩余空间也不放大。
    • flex-shrink ，定义项目缩小比例，默认值为 1，空间不足项目将缩小。
    • flex-basis ，定义分配多余空间之前，项目占主轴空间，默认值 auto ，即项目本来大小。
  • flex 默认值为 0 1 auto ， auto = (1 1 auto) ， none = (0 0 auto) 。
    • 取值一个值：无单位数字 flex-grow ；长宽百分比 flex-basis 。
    • 取值两个值：两个值无单位 flex-grow flex-shrink ；一个无单位，一个长宽百分比 flex-grow flex-basis 。

# align-self

允许单个 flex 项目有不一样的对齐方式。
取值为 auto(默认)|flex-start|flex-end|center|baseline|stretch 。

# order

定义项目在容器中排列顺序，数值越小越靠前，默认为 0。

# Grid 布局

Grid 布局将网页划分为一个个网格，通过任意组合不同的网格形成不同的网页布局。
Flex 布局是轴线布局，可以看作是一维布局。Grid 布局将容器划分了行和列，产生单元格，可以看作二维布局。
使用 display:grid 或 display:inline-grid 来指定一个 grid 容器。

# grid 容器

# grid-gap

grid-gap 属性是 grid-row-gap 与 grid-column-gap 属性是简写，用于定义网格间隙的尺寸。
grid-gap: <grid-row-gap> <grid-column-gap>
对于 grid-area 占据多个网格的情况，区域尺寸也包含 gap 的尺寸。

# place-items

place-items 是对与 align-items 与 justify-items 的简写。
place-items: <align-items> <justify-items>

  1. align-items 指定了网格元素垂直对齐方式
  1. justify-items 指定了网格元素水平对齐方式

# place-content

place-content 是对 align-content 与 justify-content 的简写。
place-content: <align-content> <justify-content> ，只有垂直或水平方向容器宽度大于网格总宽度时才生效。

  1. align-content 指定了网格元素垂直分布方式
  1. justify-content 指定了网格元素水平分布方式

# grid-auto-flow

grid-auto-flow 属性用于控制没有明确指定位置的 grid 项放置方式。

dense 关键字意为稠密堆放，当有较小网格时，会尝试寻找前方空余位置放置，使整个网格空间尽可能稠密。
此属性仅改变视觉顺序，会导致 DOM 属性与实际呈现顺序不符合，对可访问性不友好。

# 隐式网格

由 grid-template 中指定的网格称为显式网格。当在因为内容较多还需要更多的网格轨道时，网格会在隐式网格中创建行和列。
这些隐式网格的行和列的大小由 grid-auto-rows 和 grid-auto-columns 来设定。

  • number 代表起止的网格线； name 代表起止的网格线名；
  • span number 代表网格自动跨越的数量；
  • span name 代表网格会自动扩展，直到指定名称网格线。

# grid-area

grid-area 是 grid-column 与 grid-row 的缩写，同时也支持 grid-template-areas 中的 name 写法。

# place-self

place-self 是对 align-self 与 justify-self 的简写。
place-self: <align-self> <justify-self>
align-self 和 justify-self 对应 grid 容器中的 -items 属性，取值为 stretch | start | end | center ，指定单个网格元素的对齐方式。
