# 了解网页

# 网页与网站

  • 网页（webpage），即网络中的一页，它是一份适用于万维网和浏览器的文件，并且通常是 html 格式的。
  • 网站（website）是网页的集合，多个网页根据一定的规则，相互连接、组合在一起就构成了网站。

# 浏览器与浏览器内核

  • 浏览器通常指网页浏览器（web browser），是一种用于检索并展示万维网信息资源的应用程序。

  • 浏览器内核（Rendering Engine），是指浏览器最核心的部分，负责对网页语法进行解释并渲染（显示）为网页。常见的浏览器与浏览器内核如下：

    浏览器内核代表浏览器
    TridentIE 浏览器，2022 年 6 月 15 日起，微软停止对 IE 浏览器的支持
    BlinkChrome 浏览器，Edge 浏览器与 Opera 浏览器也是 Blink 内核
    webkitSafari 浏览器
    Gecko火狐（firefox）浏览器

# web 标准

Web 标准是由 W3C 和其他标准化组织制定的一系列标准的合集。W3C 是最著名的标准化组织。
标准主要包括结构 (structure)、表现 (presentation)、和行为 (behavior) 三个方面。

标准说明
结构结构用于对网页元素进行整理分类，主要指 HTML
表现表现用于设置网页元素的版式、颜色、大小等外观样式，主要指 CSS
行为行为指网页模型的定义和交互的编写，主要是 JavaScript

Web 标准的最佳体验方式：结构、样式、行为相分离。

# HTML 头部

<head> 标签内包含了所有的头部标签，在网页加载完成后， <head> 标签中的内容不会在页面中显示出来。
<head> 中可以引用脚本，引入样式表，提供元信息等。
<head> 内可以放入的标签有： <title> 、 <base> 、 <link> 、 <meta> 、 <script> 、 <style> 等。

  • <title> 定义文档的标题， <head> 内唯一必须的标签。

  • <base> 定义文档基本的链接地址 / 链接目标，文档中所有的链接都会以此为默认链接。

  • <link> 定义文档与外部资源间关系，常用于外部样式表的引入。

  • <meta> 描述了 HTML 文档的元数据，主要供搜索引擎（SEO）使用。

# 盒子标签

盒子标签没有语义，主要用于布局，结合 CSS 可以方便控制区域样式。
HTML5 在引入更多的语义标签后，应优先考虑使用有语义的标签，之后才考虑使用无语义的盒子。
<div></div> 大盒子，一行只能放置一个 div 标签。
<span></span> 小盒子，一行可以放置多个 span 标签。

# 图像标签

绝对路径：图片文件的完整路径。
相对路径：图片相对当前 HTML 页面的位置。

# 字符实体

在 HTML 中，某些字符是预留的，如大于号和小于号，它们会被浏览器误认是标签。
如果想要正确的显示这种字符，应当使用 HTML 的字符实体，常见的字符实体如下：

符号字符实体
空格&nbsp;
大于号 >&gt;
小于号 <&lt;
和号 &&amp;
人民币￥&yen;
版权 ©&copy;
注册商标 ®&reg;

完整的字符实体可以从参考手册中查到。

# 表格标签

表格用于显示、展示数据。

  • 表格属性
    属性值描述
    alignleft、center、right规定表格相对周围元素对齐方式
    border1 或者 ""规定表格单元格是否有边框，默认 ""，即没有

# 表单标签

表单由表单域、表单控件和提示信息三部分组成。

  • 表单域：包含表单元素的区域。

    属性值作用
    actionurl 地址指定接收并处理表单数据的服务器地址
    methodget/post设置表单数据的提交方式
    name名称表单域名称
    target_blank打开方式 (默认新窗口打开)

  • 表单元素：允许用户输入或者选择的内容控件。

    • input 标签：用于收集用户信息。
    属性值描述
    button定义可点击按钮
    checkbox定义复选框
    file定义输入字段和 "浏览" 按钮，供文件上传
    hidden定义隐藏的输入字段
    image定义图像形式的提交按钮
    password定义密码字段，字段中字符不显示
    radio定义单选按钮 性别：男女
    reset定义重置按钮，清除表单中的所有数据
    submit定义提交按钮，将表单中的数据发送到服务器
    text定义单行输入按钮，

    input 标签的其他属性

    属性名描述
    name定义 input 元素名 (单选按钮和多选框应有相同的 name)
    value规定 input 元素值
    checked多选框默认选中
    maxlength规定输入字段的最大长度
    size规定输入框外观长度 (用 CSS 实现)
    • label 标签：为 input 元素定义标注。
    • select 标签：实现下拉列表。
    • textarea 标签：文本域元素，多行文本框。
