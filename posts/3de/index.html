# 读书笔记 ——《费曼学习法》

# 概述

用《费曼学习法》来实践 “费曼学习法”。

当然，没有任何直接证据证明是费曼本人提出的这个说法 —— 就和老婆饼里面没有老婆一样。不过老婆饼总归还是能有一个饼，费曼学习法虽然没有费曼，但也确实是一种行之有效的学习方法。

费曼学习法的核心其实就是一点：学以致用，知行合一。即：学习与应用是互为一体的，无法对外输出的学习不能称之为学习。

是不是感觉这句话非常耳熟？不管是王阳明还是陶行知，抑或是那句 “纸上得来终觉浅，绝知此事要躬行”，表达的都是同样的意思。当然可能对费曼学习法有所耳闻的人会觉得：不对啊，费曼学习法不是 “以教代学，用输出倒逼输入” 吗，怎么到你这里就不一样了？其实不然，比起高度凝练的 “知行合一” 一词，费曼学习法更进一步指明了方向：怎么知 —— 聚焦目标，系统梳理；怎么行 —— 以教代学，输出原则。

详细一些的解释是这样说的：当学习新的知识时，尝试将自己放在传授者的立场，假设自己正在向别人讲解自己学习的内容，并思考如何用浅显易懂的语言表达出来，使得外行的听众也能明白你在说什么。在这里，输出即是对知识的应用，也是对知识的检验，试想：如果我们对所学的知识都不了解明白，又怎么能把它好好地向其他人表述出来呢？

实践费曼学习法的途径可以参考以下五步：确立目标，理解对象，以教代学，回顾反思，简化吸收。

# 确立目标

确立目标和聚焦目标是学习的第一步。

我们开始一个学习计划的目的有很多，但是与读书相似，大抵也可以划分为功利性的与非功利性的。虽然《费曼学习法》的著书者推崇的是非功利性的学习 —— 学习应该是开心的有趣的或者说是符合个人兴趣的，这没有错。但是在生活中，我们确实有不少需要功利性的学习的情况。这种时候，明确学习的目标，可以帮助我们把有限的精力集中在 “应该做” 的事情上，让我们的思维更加清晰，行动也更加有针对性，也就是 “好钢要用在刀刃上” 的道理。

对于目标的确立，SMART 原则提供了一种简单明了的方法：

  1. 明确的（specific）—— 目标必须是明确具体的。
  2. 量化的（measurable）—— 目标必须是可以量化评估的。
  3. 可达到的（achievable）—— 目标是能力范围内可以达到的。
  4. 成就感的（rewarding）—— 目标能带来成就感。
  5. 时限性的（time-bound）—— 目标必须有完成的期限。

举例来说，“这周内阅读完成《费曼学习法》，并运用‘费曼学习法’撰写这本书的读书笔记” 就是一个较为清晰合理的目标。

当然，大目标还可以被拆分为小目标，目标和计划也会随着学习过程中的反馈来进行调整。因此列出目标是很重要的，对学习阶段的正确规划并且按照学习进度进行调整改善，也是同样需要时间来做的。

# 理解对象

确定了学习的目标，下一步就是要去理解要学习的对象。我们要彻底深入地理解所要学习的对象，用系统性的框架将知识点组合起来。其实这也是大部分教材在第一章都会讲的类似 “绪论” 的东西。就如同建房子之前会先有施工图纸一般，框架的搭建能让我们明白我们将要学习的是什么，在这个过程中，我们也完成了对知识的一个筛选，保留了我们需要的、可靠的、重要的知识。

在这里，可以用到的工具是思维导图。思维导图是一种知识的可视化，有助于以 “鸟瞰” 的方式把握知识的系统结构。一方面，它把文字碎片化的知识用流程图的方式串连了起来，有助于知识的系统化；另一方面，它也有助于形成高质量的学习笔记，并为之后的输出做好了准备。

# 以教代学

输出，一个简单的办法就是复述你所学的知识并让听者理解。这是实践 “知行合一”，并转变为 “主动学习” 的第一步，也是最重要的一步。

首先是凭借印象复述，这个过程中包含了对知识的回顾和整理；然后是在复述中提出问题，这个过程是对知识的思考与分析；最后是在复述中加入自己的观点，这是把知识与自身结合后得到的见解与领悟。

输出的过程，一方面是通过复述，加深了对知识的印象；另一方面也是加深了对知识的理解。输出的过程，也是对所学知识的检验：通过与别人的交流与沟通，我们得以发现记忆中的偏差，对错误的纠正，也进一步加深了对知识的理解。

# 回顾反思

在复述完成后，检查、审视并总结复述的内容，确保我们阐述内容的正确性。对于怀疑不定的内容，要在现实中找到一个坚固的桥梁，确保它不是一个似是而非的空中楼阁 —— 因为知识最终是要融入到我们的生活中的，而不是裱起来欣赏的一个玩物。对发现错误的内容，则要重新回顾知识库，重新理解它。

反思则不同于回顾总结，它要求用一种解构的方式去看待我们已经学习的内容，确保学习的是正确的知识。因为知识本身也就是在动态变化的，所有知识都存在局限性，权威的知识也只是在当下的环境和条件中是正确的。随着环境和条件的变化，知识也会进行更新变化，这就需要反思的途径使知识保存常新。

# 简化吸收

有句话叫 “吾生也有涯，而知也无涯”，这句话的后半句是 “以有涯随无涯，殆已”。通俗的讲法就是，食物吃多了还会消化不良，知识也是同样的。

更何况，世界上那么多的信息，我们不可能尽收脑海中。学习完成之后，为了确保我们学习到的是知识，而不是一堆庞杂的信息，还需要的是对所学内容成体系地简化吸收，最终将知识变成 “自己的”。有一种说法就是 “把薄书读厚，再把厚书读薄”，“把书读厚” 的过程是知识的记忆与理解，“把书读薄” 就是理解后的消化吸收。

简化吸收也应该紧紧围绕着学习的目的 —— 学以致用，把当下需要的、重要的、核心的知识，通过简化吸收，内化为自己的，从而构建起自己的知识体系。

# 小结

书籍本身围绕费曼学习法提出了一套指导性的学习方法。费曼学习法的核心其实就是以通俗易懂的方式向他人讲解自己正在学的知识，而这个过程本身就是学以致用，知行合一的一种体现。通过这样的方式，能更好的检查知识的理解掌握程度，达到更好的学习效果。
