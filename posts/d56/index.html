# 题目描述

题目来源：LeetCode

给你两个非空的链表，表示两个非负的整数。它们每位数字都是按照逆序的方式存储的，并且每个结点只能存储一位数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

# 解题思路

# 链表

这道题首先需要理解的是链表的数据结构。
链表是一种线性的数据结构，它由一个指向该链表的指针（头指针）和一系列的存储单元（结点）组成。

flowchart LR
id1((data1))-->id2((data2))-->id3((date3))-->null

利用 JavaScript 来模拟实现的链表可能是这样的：

知道了链表的数据结构后，此题的解法就是遍历链表，并将遍历的值求和，建立一个新链表。
时间复杂度：O(max(n,m))O(max(n,m))O(max(n,m))，nnn 和mmm 分别是两个链表的长度。
空间复杂度：O(1)O(1)O(1)

# 注意事项

  • 题中链表为逆序，给出链表也需要是逆序，因此需要使用尾插法创建。
  • 两数求和需要注意进位问题。
