# 题目描述

题目来源：LeetCode

给定一个字符串 s ，请你找出其中不含有重复字符的最长子串的长度。

# 解题思路

这道题目按照官方或者说有刷题经验的标准解法应该是滑动窗口，当然我一开始是没有很清晰的思路的……
既然返回是最长无重复子串的长度，我的想法是：

  • 声明两个变量，一个用来存储子串长度，一个用来存储开始位置序号；
  • 使用字符串的 slice 方法获取子串，然后判断子串中是否有重复字符；
  • 若没有重复字符，将无重复子串的长度 + 1，重复获取子串的步骤；
  • 若有重复字符，将起始位置序号移动，然后再重新获取子串判断。

# 官方题解

滑动窗口的思路会更清晰一些：
用两个指针判断子串的左右边界，维护一个哈希集合判断是否有重复字符。
当没有重复字符时，移动右指针，并将新的字符加入哈希集合；当有重复字符，移动左指针，并将这个字符从哈希集合中删除。
例如：

  • ababc -> (a) babc -> (ab) abc -> a (ba) bc -> ab (ab) c -> ab (abc)

感觉官方的思路上还可以有所优化：

  1. 是否借鉴自己实现的思路，左指针并不是指移动一格，而是直接移动到重复字符的后一格？
  2. 左指针移动到重复字符后一格后，右指针直接移动到 left+max+1 格（长度比 max 小的不可能出现更长的子串）？
