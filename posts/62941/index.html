# CSS 单位

CSS 中的单位可以分为两种：绝对单位和相对单位。

  1. 绝对单位
绝对单位说明等价代换
cm厘米1cm = 96px/2.54
mm毫米1mm = 1/10th of 1cm
in英寸1in = 2.54cm = 96px
pt点1pc = 1/6th of 1in
px像素1px = 1/96th of 1in
  • 绝对单位一般用于传统平面印刷，实际网页开发中使用较少。
  • px 像素值在MDN中被归类为绝对单位。
  1. 相对单位
相对单位说明
em在 font-size 中是相对于父元素字体大小，在其他属性中使用是相对自身字体大小
rem相对根元素字体大小
%相对父元素的百分比
vw相对可视区域宽度的百分比
vh相对可视区域高度的百分比

# CSS 精灵图

精灵图 (CSS Sprites)，又称 CSS 雪碧图。主要使用场景为页面的背景。
目的：减少服务器接收和发送请求的次数，提高页面加载速度。
实现：将多张小图片拼合成一张大图片，使用时通过 background-position 来展示需要使用的那部分图片。

缺点：使用较麻烦，每一种图片都需要分别设置并调整位置。

# 表单样式

  • 轮廓线 (outline)：

    • outline 与 border 很类似，也有 outline-style 、 outline-width 与 outline-color 属性。
    • outline 不占据空间，绘制与元素四周。默认情况下 outline 用于 :focus 样式。
    • 使用 outline:0 或者 outline:none 去除默认的 outline 。

  • resize 属性

    • resize 属性指定一个元素，允许用户调整大小。
    • 禁止拖拽调整元素尺寸 (如 textarea ) 使用 resize:none 。

  • 列表项符号 (list-style-type)

    • ol 可取值： decimal | lower-roman | upper-roman | lower-alpha | upper-alpha
    • ul 可取值： disc(实心圆) | circle(空心圆) | square(正方形) | url(自定义图片)
    • 去除默认值： list-style-type:none

  • user-select 属性
    user-select 属性用来控制用户能否选中文本。

    属性值说明
    none元素及其子元素的文本不可选中
    all包含该元素的最顶层元素也会被选中
    text用户可以选择文本
    contain允许在元素内选择，但是选区限制在该元素的边界之内
    auto取决于父元素的 user-select 和元素本身情况
