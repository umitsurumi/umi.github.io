# CSS 标准

CSS3 是最新的 CSS 标准，从 CSS3 开始，CSS 被划分为众多模块，这些模块彼此独立并按照各自进度来进行标准化。

# 选择器

# 属性选择器

属性选择器在 CSS2 中引入，在 CSS3 中得到了进一步的扩展，它的使用语法如下：

选择器说明
[attr]选中具有 attr 属性的元素
[attr=val]选中具有 attr 属性，且值为 val 的所有元素
[attr~=val]选中具有 attr 属性，且值包含 val 的所有元素
[attr | =val]选中带有以 val 开头的 attr 属性的所有元素
[attr^=val]选中 attr 属性值以 val 开头的所有元素
[attr$=val]选中 attr 属性值以 val 结尾的所有元素
[attr*=val]选中 attr 属性值包含子串 val 的所有元素

# 伪类与伪元素

除了 :nth-of-child(n) 与 nth-of-type(n) 系列，CSS3 还提供了几个较为常用的新选择器：

  1. :root
    :root 选择器匹配文档根元素，在 HTML 中则始终是 html 元素，配合 CSS 变量可以在文档中重复使用。

  2. ::after 与 ::before
    在 CSS3 规范中，用 : 和 :: 来对伪类和伪元素进行区分。
    ::after 表示在选中元素之后插入内容， ::before 表示在选中元素之前插入内容，两者必须搭配 content 来进行使用。
    优点：减少 DOM 节点数，优化页面结构。
    缺点：不利于 SEO，无法使用 js 控制。

# 盒子属性

# box-sizing

CSS3 中可以通过 box-sizing 属性来指定盒子模型。

  • box-sizing: content-box ，默认值，标准盒子模型。
  • box-sizing: border-box ，IE 盒子模型，又称怪异盒子。
    它们区别如下图所示：

在标准盒子中，盒子宽度 = width + padding + border ;
在怪异盒子中，盒子宽度 = width ，而内容区空间会被 padding 和 border 挤进去。

# box-shadow

box-shadow 用于给元素添加阴影效果。

类似的，使用 text-shadow 也可以给文本添加阴影：
text-shadow: x偏移 y偏移 模糊半径 颜色

# border 属性

CSS3 的边框新增了 border-image 与 border-radius 属性。

  • border-image 属性用来设置图像作为围绕元素的边框。

    属性取值说明
    border-image-source路径选择图片路径作为绘制边框来源
    border-image-slice像素值规定图像边框图像的内偏移
    border-image-width图像宽度多个值时应用顺序与内外边距相同
    border-image-outset像素值定义边框图像可超出边框盒的大小
    border-image-repeatstretch repeat round space拉伸 / 平铺 / 缩放平铺 / 空白填充平铺

    border-image 可以进行简写：

    • 顺序为 url() slice width outset repeat
    • 默认值为 none 100% 1 0 stretch

  • border-radius

    • border-radius: 30px 圆角； border-radius: 30px/20px 椭圆角。
    • border-radius 可以给所有元素设置圆角，当设置 border-radius: 50% 时，就得到了一个圆。
    • border-radius 实际上是指定了一个圆 (或椭圆) 与外边框进行交集，从而形成的圆角效果。
    • border-radius 也是对四个边角的简写，同样也可以四个角分别指定。

# 使用网络字体

# 背景设置

  • background-image 可以设置多个图片，显示在最顶层的为第一张。
  • background-siez 可以为图片指定大小，关键词有 contain|cover 。
  • background-origin 指定图片位置，关键词 border-box|padding-box(默认)|content-box 。
  • background-clip 指定图片绘制区域，关键词 border-box(默认)|padding-box|content-box 。

# 过渡与动画

# 过渡 (transition)

transition 属性允许在不使用脚本或添加动画的情况下，平滑地改变属性值。
要应用过渡属性，需要指定添加效果的属性，并且指明效果持续时间。

属性值说明
transition-property规定过渡效果应用的 CSS 属性
transition-duration规定过渡效果持续时间 (秒或毫秒)
transition-delay规定过渡效果的延迟 (秒)
transition-timing-function规定过渡效果速率曲线

类似的， transition 也可以对这四个属性进行简写:

  • 只有具有中间值的属性才可以应用过渡效果，如 display 属性不能实现过渡。

  • transition- 属性是即时生效的，如果写在 :hover 里面，则移入有效果，移出无效果。

  • transition-timing-function 各关键词参考见MDN。

    关键词说明
    linear线性应用过渡效果
    ease由快到慢，逐渐变慢
    ease-in速度越来越快
    ease-out速度越来越慢
    ease-in-out先加速后减速
    cubic-bezier(n,n,n,n)在 cubic-bezier 函数中定义效果

# 动画 (animation)

动画可以设置多个节点来控制一个或一组动画效果。相较过渡，动画可以实现更多变化、更多控制、自动播放等。

  • 动画不需要事件触发，过渡需要。
  • 过渡只有一组 (开始与结束) 关键帧，动画可以设置多个。

同样， animation 也可以对动画属性进行简写:
animation: name duration timing-function delay iteration-count direction fill-mode play-state
